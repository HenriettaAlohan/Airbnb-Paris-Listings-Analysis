# Airbnb Paris Listings Analysis

## 📊 Project Overview
This project analyzes Airbnb listings in Paris to extract insights on pricing trends, host behavior, and neighborhood distribution. The goal is to turn raw listing data into actionable insights using Python data analysis techniques.

## 🎯 Objectives
- Analyze pricing patterns across Paris neighborhoods
- Study host activity and listing distribution
- Identify demand and availability trends
- Visualize key insights for better decision-making

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
- Exploratory Data Analysis (EDA)

## 📈 Key Insights
- Pricing varies significantly across districts in Paris
- Some neighborhoods dominate listing volume
- Host distribution is uneven across the city
- Clear trends exist in pricing and availability patterns
